# Todo App

<br>

## Development

To install and work on `https://github.com/halilatilla/todo-app`

```bash
git clone https://github.com/halilatilla/todo-app
cd todo-app
npm install or yarn install
```

<br>

```bash
yarn dev
```

`http://localhost:3000/`
